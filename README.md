# TemplateGJ
Project template for prototype or game jam.

By default in 2D mode.

For Unity 2019.2.15.

## TODO Readme
After using this template, you need to:
- Put team credits.
- Write a project description.
- Link to the game itch.io page.
- List all tools used.
- (Optional) Insert a logo picture.
